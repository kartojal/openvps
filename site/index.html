How it works

1

Request

POST /v1/provision with desired specs. Get a 402 Payment Required with pricing.

2

Pay

Transfer stablecoins on Base, Celo, or Tempo. x402 or MPP — your choice.

3

Compute

Get a public IP, SSH port, and private key. Root access to Ubuntu 24.04.

# 1. Request a VPS (get payment challenge) curl -s https://openvps.sh/v1/provision \ -H "Content-Type: application/json" \ -d '{"vcpus": 2, "ram_mb": 1024, "duration": 3600}' # Response: 402 Payment Required # { # "amount": "12548", ← $0.01 in microdollars # "recipient": "0x...", ← pay here # "chain_id": 8453 ← Base # } # 2.
